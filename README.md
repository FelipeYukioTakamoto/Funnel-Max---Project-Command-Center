# 📊 Funnel Max - Project Command Center (Protótipo)

Este repositório contém um protótipo de *dashboard* interativo desenvolvido como material complementar para o Teste Técnico de **Analista de Projetos** da Funnel Max. 

O objetivo deste painel é tangibilizar as soluções propostas no documento principal do teste, demonstrando uma visão orientada à resolução de problemas, monitorização visual e prevenção de gargalos.

## 🚀 Aceder ao Protótipo Online

Pode visualizar o Command Center a funcionar diretamente no seu browser através do GitHub Pages:
👉 **[Clique aqui para aceder ao Dashboard Live](https://felipeyukiotakamoto.github.io/Funnel-Max---Project-Command-Center/)**

---

## 🎯 Principais Funcionalidades Mapeadas

1. **Visão de Gargalos em Tempo Real (Caminho Crítico):** Identificação visual imediata do status da operação técnica. Destaca o bloqueio atual (dependência do Produtor Parceiro) e o tempo de SLA restante para não comprometer o projeto.

2. **Formulário de Onboarding (O "Pedágio"):** Uma simulação interativa de um processo de entrada de novos parceiros. Garante que a equipa interna só inicie os trabalhos de *Copy* e *Infraestrutura* após a submissão do link final e da validação comercial (comissão), evitando desperdício de horas operacionais.

3. **Análise de Validação de Oferta (Gráfico Interativo):** Visualização baseada em dados reais vs. projeções para sustentar decisões de negócio. Ilustra a tese defendida no teste: a validação de uma oferta deve ser pautada no **EPC** (Earning Per Click) após a injeção de tráfego, e não descartada precocemente apenas por uma Taxa de Abertura inicial de 12%.

---

## 🛠️ Tecnologias Utilizadas

Para manter a simplicidade e leveza, este protótipo foi construído sem *frameworks* complexas (conceito *Single Page Application*):
* **HTML5 & CSS3:** Estrutura e design responsivo (estilo "Dark Mode" para foco em dados).
* **JavaScript (Vanilla):** Lógica de interação do formulário.
* **Chart.js:** Biblioteca utilizada para a renderização dinâmica do gráfico de métricas.
